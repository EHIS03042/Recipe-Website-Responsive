## Project Overview

This project is a **responsive recipe website** built with a **mobile-first approach** and dynamically populated using data from [TheMealDB API](https://www.themealdb.com/api.php). It allows users to browse meals (e.g., chicken recipes), view ingredients, cooking instructions, and watch recipe videos on YouTube.

Designed with both **usability** and **responsiveness** in mind, this application is optimized for mobile and scales smoothly to larger screens.

##  Key Features

- **Mobile-First Responsive Layout**  
  Uses CSS Grid and Flexbox to create a seamless layout across all device sizes.

- **Dynamic Meal Data Rendering**  
  Fetches JSON data from TheMealDB API and dynamically displays:
  - Meal image
  - Meal title
  - Cooking instructions
  - Top 5 ingredients
  - YouTube link for cooking video

- **Concise and Accessible UI**  
  Minimalist layout with clear typographic hierarchy and intuitive structure.


## Technologies Used

- **HTML5** – Semantic markup
- **CSS** – Responsive layout using Grid and media queries
- **JavaScript (ES6+)** – API fetch and DOM rendering
- **[TheMealDB API](https://www.themealdb.com/api.php)** – JSON meal data source
